## Features

* Zweiter Spannungsregler für unsaubere Verbraucher
    * DC/DC-Wandler für Zenerdiodentest bis 30V
    * PWM-geregelte LCD-Hintergrundbeleuchtung

* Passt in Strapubox 6000
